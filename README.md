# python-for-vps
专注制作贴合VPS使用的一键脚本，敬请期待！

这里提供的python脚本都是在自己的电脑上运行的，需要现在本机安装python环境安装paramiko库。
就可以用现实电脑操作控制所有VPS了，包括批量更新，查看所有VPS情况，批量安装docker，批量上传文件，批量测速等操作！
